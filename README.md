# personal_website

This is a simple one-page website.

# Deploy

1) install docker (https://docs.docker.com/engine/install/ubuntu/)
2) sudo apt install python3-pip
3) sudo pip3 install docker-compose
4) git clone https://github.com/sentenza-dev/personal_website.git (sudo apt install git)
5) add .crt and .key files for https
6) sudo docker-compose up -d --build
